# Panduan Pengguna GENY-StuntCare

Panduan lengkap untuk menggunakan aplikasi GENY-StuntCare.

## Registrasi & Login

### Cara Membuat Akun

1. Buka aplikasi GENY-StuntCare
2. Klik tombol **"Mulai Sekarang"** di halaman utama
3. Masukkan email dan password yang kuat
4. Verifikasi email Anda melalui link yang dikirimkan
5. Akun siap digunakan!

### Login ke Aplikasi

1. Klik **"Login"** di halaman utama
2. Masukkan email dan password
3. Klik **"Masuk"**

:::tip Tips
Gunakan password yang kuat dengan kombinasi huruf besar, kecil, angka, dan simbol.
:::

## Dashboard Utama

Dashboard adalah halaman pertama yang Anda lihat setelah login. Di sini Anda dapat:

### Informasi Ringkas

- Status kesehatan anak Anda
- Grafik pertumbuhan terbaru
- Notifikasi penting

### Menu Navigasi

- **Profil Anak** - Kelola data anak Anda
- **Monitoring** - Lihat riwayat pertumbuhan
- **Rekomendasi** - Dapatkan saran dari AI
- **Chat** - Konsultasi dengan ahli
- **Pengaturan** - Ubah preferensi akun

## Profil Anak

### Menambah Anak Baru

1. Buka menu **"Profil Anak"**
2. Klik tombol **"+ Tambah Anak"**
3. Isi formulir dengan data anak:
   - Nama lengkap
   - Tanggal lahir
   - Jenis kelamin
   - Nama ibu
4. Klik **"Simpan"**

### Edit Profil Anak

1. Pilih anak yang ingin diedit
2. Klik tombol **"Edit"**
3. Ubah data yang diperlukan
4. Klik **"Simpan Perubahan"**

## Monitoring Pertumbuhan

### Input Data Antropometri

Data antropometri adalah pengukuran fisik anak yang penting untuk monitoring pertumbuhan.

#### Cara Input Data:

1. Buka menu **"Monitoring"**
2. Klik **"+ Input Data Baru"**
3. Pilih anak yang akan dimonitor
4. Isi data pengukuran:
   - **Berat Badan** (dalam kg)
   - **Tinggi Badan** (dalam cm)
   - **Lingkar Kepala** (dalam cm) - untuk anak < 5 tahun
   - **Lingkar Lengan Atas** (dalam cm)
5. Klik **"Simpan"**

### Lihat Grafik Pertumbuhan

1. Pilih anak dari daftar
2. Sistem akan menampilkan:
   - Grafik pertumbuhan berat badan
   - Grafik pertumbuhan tinggi badan
   - Tren pertumbuhan (naik/stabil/menurun)
   - Perbandingan dengan standar WHO

### Riwayat Data

Semua data pengukuran disimpan dan dapat Anda lihat kapan saja dalam bentuk:

- Tabel riwayat
- Grafik visual
- Laporan PDF (dapat diunduh)

## Rekomendasi AI

### Cara Mendapatkan Rekomendasi

1. Pastikan data anak Anda sudah terupdate
2. Buka menu **"Rekomendasi"**
3. Sistem AI akan menganalisis:
   - Data pertumbuhan
   - Tren kesehatan
   - Faktor risiko stunting
4. Anda akan menerima:
   - **Risiko Assessment** - Tingkat risiko stunting
   - **Rekomendasi Nutrisi** - Saran makanan sehat
   - **Tips Perawatan** - Panduan perawatan anak
   - **Jadwal Checkup** - Waktu pemeriksaan berikutnya

## Chat Support

### Fitur Chat

Berkonsultasi langsung dengan ahli kesehatan anak dan gizi.

#### Cara Menggunakan Chat:

1. Buka menu **"Chat"**
2. Klik **"+ Chat Baru"**
3. Pilih topik:
   - Pertanyaan Kesehatan
   - Nutrisi Anak
   - Perkembangan Anak
   - Lainnya
4. Tulis pertanyaan Anda
5. Tunggu respons dari ahli

### Jam Operasional

- **Senin - Jumat:** 08:00 - 18:00 WIB
- **Sabtu:** 08:00 - 14:00 WIB
- **Minggu:** Tutup

:::warning Catatan
Untuk kasus darurat, hubungi unit kesehatan terdekat.
:::

### Riwayat Chat

Semua percakapan disimpan sehingga Anda dapat:

- Melihat kembali saran yang diberikan
- Melanjutkan percakapan sebelumnya
- Membagikan chat dengan tenaga kesehatan lain
